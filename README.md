# How to Run the Project locally

## Frontend
### 1. Go to the frontend folder
`cd frontend`

### 2. Install the dependencies

`npm install`

### 3. Run the project

`npm run dev`
`http://localhost:5173/`

## Backend(server)
### 1. Open the server folder
`cd server`
### 2. Install the dependencies
`npm i`
### 3. Start the server
`npm start`
`http://localhost:3000/`
